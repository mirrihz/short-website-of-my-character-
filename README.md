# Short Website of My Character

Сайт-портфоліо, присвячений моєму оригінальному персонажу King D. Mirri зі світу One Piece.

## 📝 Про проект
Ця сторінка створена українською мовою та містить опис, історію та характеристики персонажа. 

## 🛠 Технології
* HTML5
* CSS3 (якщо є)

## 🚀 Як переглянути
